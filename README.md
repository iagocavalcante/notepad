# Notepad

A note-taking app built with Ionic and SQLite, inspired by Evernote behaviors.

## Tech Stack

- **Framework:** Ionic 6
- **Frontend:** Angular 16
- **Database:** SQLite (cordova-sqlite-storage)
- **Storage:** @ionic/storage

## How to Run

### Prerequisites

- Node.js 18+
- npm 9+
- Ionic CLI: `npm install -g @ionic/cli`

### Install Dependencies

```bash
npm install
```

### Run in Browser

```bash
npm start
# or
ionic serve
```

### Build for Android

```bash
ionic capacitor add android
ionic capacitor sync
ionic capacitor run android
```
